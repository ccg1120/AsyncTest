1. **NET FrameWork 4.6 적용**

- Task를 사용하기위해서는 4.5이상의 .net FrameWork가 필요하다. 설정은 Editor -> Project setting -> Player -> Scripting Runtime Version 을 .NET 4.6으로 적용 하면 Unity가 다시 시작된다.
